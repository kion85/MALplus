# MAL+

Команда	Действие
---------	----------
/regs	Показать текущее состояние всех регистров, флагов и счетчика команд
/step	Выполнить ровно 1 такт / одну команду программы из памяти
/run	Запустить полное выполнение программы из памяти до команды STOP (8x3)
/set R0 ЧИСЛО	Записать вручную любое значение от 0 до 255 в регистр R0
/reset	Полностью сбросить состояние виртуальной машины до начального
/exit	Закрыть интерпретатор

/+ ЧИСЛО	 Сложить: прибавить указанное число к основному регистру R0
/- ЧИСЛО 	Вычесть: отнять указанное число от регистра R0

Компиляция через GCC/G++
g++ malplus_vm.cpp -O0 -o mal

Запуск
./mal
